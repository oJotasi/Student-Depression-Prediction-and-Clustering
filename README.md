# Student-Depression-Prediction-and-Clustering

## Descripción
Pipeline de aprendizaje automático para la predicción del riesgo de depresión en estudiantes y el análisis de agrupamientos de comportamiento utilizando EDA, Random Forest, XGBoost y K-Modes.

---

## Descripción general
Este proyecto aplica técnicas de Machine Learning para analizar y predecir el riesgo de depresión en estudiantes universitarios. Combina análisis de datos, clustering y modelos de clasificación para identificar factores clave asociados a la salud mental.

---

## Métodos
- Análisis Exploratorio de Datos (EDA)
- Preprocesamiento de datos y selección de características
- Clustering (K-Modes)
- Modelos de clasificación:
  - Random Forest
  - XGBoost

---

## Resultados
- Accuracy: ~83%
- XGBoost muestra mejor desempeño en la detección de estudiantes de alto riesgo

### Factores clave:
- Pensamientos suicidas
- Estrés académico y financiero
- Sueño y hábitos de vida
